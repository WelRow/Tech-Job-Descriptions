# Tech Job Descriptions Library

Created to group similar Tech job descriptions, that has distinctive job titles into generalized, well known job titles, revealing patterns in minimum requirements so you'll knwo to learn when applying for jobs.

### These tech Job Descriptions are sorted by [UNCC CS B.S. Concentrations](https://cci.charlotte.edu/student-life/cci-advising-center/undergraduate-cs-concentrations-and-degree-plans-study/), then by niche areas in their concentrations.

### They will have different titles, but they all do the same type of work.
